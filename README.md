 Project Description

The Real-Time Chat Application Backend is a server-side system built to support secure, fast, and real-time communication between users.

It enables users to:
- Register and authenticate securely
- Add contacts using email IDs
- Send and receive real-time messages
- Automatically create conversations when a message is received
- Maintain persistent chat history

The backend uses Socket.IO for real-time bi-directional communication and REST APIs for authentication, user management, and contact handling.

JWT-based authentication ensures secure access to protected routes, while MongoDB stores users, contacts, and messages efficiently.

-------------------------------------------------------------------------------------------------------------------------------------------------

 Technologies Used

- Node.js – Server-side runtime
- Express.js – REST API framework
- MongoDB & Mongoose – Database and data modeling
- Socket.IO – Real-time messaging
- JWT (jsonwebtoken) – Authentication & authorization
- bcryptjs – Password hashing
- express-validator – Input validation
- Helmet – Security headers
- CORS – Cross-origin communication
- dotenv – Environment configuration
- Nodemon – Development auto-reload
